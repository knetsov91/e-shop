<h1>E-Shop project</h1>

<h2>Overview</h2>
<p>Project is build by using microserivce architecure.
Each microservice is implemented as Spring Boot project.
</p>

<h2>Tech Stack</h2>
<ul>
    <li>Java 21</li>
    <li>Spring Boot 3</li>
    <li>Kafka</li>
    <li>PostgreSQL</li>
    <li>MongoDB</li>
    <li>Docker</li>
    <li>Traefik</li>
    <li>HashiCorp Consul</li>
    <li>Prometheus</li>
    <li>Grafana</li>
    <li>Elasticsearch</li>
    <li>Kibana</li>
</ul>
