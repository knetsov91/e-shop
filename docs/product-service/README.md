<h1>Product microservice</h1>
<h2>Overview</h2>
<p>This microservice is implemented using Spring Boot Java. 
</p>
<h2>Tech Stack</h2>
<ul>
    <li>Java 21</li>
    <li>Spring Boot 3</li>
    <li>PostgreSQL</li>
</ul>

## Endpoints

- `GET /api/v1/products` (Public) — List all products
- `POST /api/v1/products` (Protected) — Create a new product
